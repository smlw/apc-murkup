Подтверждение прав на http://domain-name.ru

  • Мета-тег
  • HTML-файл
  • JSON-файл

Добавьте в код главной страницы вашего сайта (в раздел head) мета-тег:

<meta name="yandex-verification" content="26773ea554023e11" />

Зайдите на главную страницу сайта и убедитесь, что мета-тег появился в html-коде страницы. В большинстве браузеров это можно сделать выбрав пункт «Исходный код страницы» в контекстном меню. На некоторых сайтах обновление мета-тегов может занимать несколько минут! Нажмите кнопку «Проверить» после того, как мета-тег появится.


В корне сайта создайте файл с именем yandex_26773ea554023e11.html и со следующим содержимым:

<html> <head> <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"> </head> <body>Verification: 26773ea554023e11</body> </html>

Убедитесь, что файл по адресу http://www.apc.ru/yandex_26773ea554023e11.html открывается


В корне сайта создайте файл с именем yandex_26773ea554023e11.json и со следующим содержимым:

{ "domain": "http://apc.ru", "secret": "random string" }

Убедитесь, что файл по адресу http://www.apc.ru/yandex_26773ea554023e11.json открывается
